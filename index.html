📊 EVM — Kazanılmış Değer Metrikleri
💜
BCWP — Kazanılmış Değer
Budgeted Cost of Work Performed / EV
TANIM
Tamamlanan işin bütçesel değeri
%80 tamamlanmış 100M TL'lik iş → BCWP = 80M TL. Gerçek harcamadan bağımsız.
BCWP = Σ (Bütçe × Fiziksel İlerleme%)
İpucu: BCWP > ACWP → tasarruf. BCWP < ACWP → fazla harcama.
💙
BCWS — Planlanan Değer
Budgeted Cost of Work Scheduled / PV
TANIM
Bu tarihe kadar yapılması planlanan işin bütçesel değeri
Takvim ilerlemesine göre bugüne kadar tamamlanmış olması gereken iş miktarı.
BCWS = Σ (Bütçe × Takvim İlerleme%)
İpucu: BCWP < BCWS → proje takvim gerisinde.
❤️
ACWP — Gerçek Maliyet
Actual Cost of Work Performed / AC
TANIM
Bugüne kadar gerçekte harcanan tutar
Veri girişindeki "Gerçekleşen" sütunundan gelen toplam. Faturalar, ödemeler, fiili giderler.
ACWP = Σ Gerçekleşen Harcamalar
🟡
CPI — Maliyet Performans Endeksi
Cost Performance Index
FORMÜL
Harcanan her 1 TL'ye karşılık üretilen iş değeri
CPI = BCWP ÷ ACWP
CPI > 1 → Bütçe altında ✅ CPI = 1 → Tam bütçede 🟡 CPI < 1 → Bütçe aşımı 🔴
Örnek: CPI = 0.85 → Her 100 TL harcamada 85 TL'lik iş yapılıyor. Proje bütçesi ~%18 aşacak.
🩵
SPI — Takvim Performans Endeksi
Schedule Performance Index
FORMÜL
Planlanan işe göre gerçekte ne kadar tamamlandığı
SPI = BCWP ÷ BCWS
SPI > 1 → Takvim ilerisinde ✅ SPI = 1 → Tam takvimde 🟡 SPI < 1 → Takvim gerisinde 🔴
Örnek: SPI = 0.90 → Planın %90'ı tamamlanmış, bu aydaki işin %10'u sarkıyor.
💚
CV — Maliyet Sapması
Cost Variance
FORMÜL
Kazanılan değer ile gerçek harcama arasındaki fark
CV = BCWP − ACWP
CV > 0 → Tasarruf ✅ CV < 0 → Aşım 🔴
Örnek: CV = −5M TL → Yapılan işin bütçesel değerinden 5M TL fazla harcanmış.
📅
SV — Takvim Sapması
Schedule Variance
FORMÜL
Kazanılan değer ile planlanan arasındaki fark (para cinsinden)
SV = BCWP − BCWS
SV > 0 → Takvim ileride ✅ SV < 0 → Takvim geride 🔴
🔮
EAC — Tamamlanma Tahmini
Estimate At Completion
FORMÜL
Mevcut performans devam ederse projenin toplam maliyeti
EAC = Toplam Bütçe ÷ CPI
Örnek: Bütçe 100M, CPI = 0.90 → EAC = 111M TL. Proje 11M TL fazlaya mal olacak.
⚖️
VAC — Tamamlanma Sapması
Variance At Completion
FORMÜL
Toplam bütçe ile EAC arasındaki fark — nihai tasarruf veya aşım
VAC = Toplam Bütçe − EAC
VAC > 0 → Tasarrufla tamamlanacak VAC < 0 → Aşımla tamamlanacak
📈 S-Eğrisi — Nasıl Okunur?
📐
Planlanan Eğri (Mavi)
Takvime göre beklenen kümülatif ilerleme
Her PYP'nin başlangıç/bitiş ayı ve ağırlığından hesaplanır. Başlangıçta yavaş, ortada hızlı, bitişe doğru yavaşlayan "S" formunu alır.
Dikkat: Doğru plan eğrisi S şeklinde olmalıdır. Düz veya erken dik çıkan plan gerçekçi değildir.
✅
Gerçekleşen Eğri (Yeşil)
Fiili fiziksel ilerlemenin kümülatif değeri
Alt PYP veya Ana PYP fiziksel ilerleme değerlerinden ağırlıklı hesaplanır. Sadece mevcut aya kadar gösterilir.
Okuma: Yeşil mavi üstündeyse → takvim ilerisinde ✅ Mavi altındaysa → takvim gerisinde 🔴

🎨 Renk ve Durum Kodları
🟢
Yeşil — İyi Performans
• Fiziksel ilerleme ≥ %80
• CPI ≥ 1.0 → Bütçe altında
• CV > 0 → Tasarruf
• Risk: İyi
🟡
Sarı — Dikkat Gerekli
• Fiziksel ilerleme %50–79
• CPI 0.85–1.0 → Hafif baskı
• CV küçük negatif → İzleme
• Risk: Orta / Yüksek
🔴
Kırmızı — Kritik Durum
• Gerçekleşen > Bütçe × 1.25
• CPI < 0.85 → Ciddi aşım
• CV büyük negatif → Müdahale
• Risk: Kritik
